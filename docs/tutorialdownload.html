Configurando o PowerShell

Abra o PowerShell como administrador e verifique a versão do Python:

python --version

Se não estiver instalado, baixe e instale diretamente com:

winget install -e --id Python.Python.3

Certifique-se de que o PowerShell permite scripts:

Set-ExecutionPolicy Unrestricted

Confirme com A e verifique com:

Get-ExecutionPolicy

Criando Ambiente Virtual e Baixando o Programa

Recomenda-se criar a venv em uma pasta dedicada para seus projetos ou documentos. Navegue até o local desejado no terminal com o comando:

cd <caminho_da_pasta>

Em seguida, crie a venv com:

python -m venv rankadmet

E ative o ambiente virtual com:

.\rankadmet\Scripts\activate

Acesse esse repositório no GitHub. Na seção "Assets" da última versão do RankAdmet, clique com o botão direito no arquivo .whl e copie o link. Em seguida, use o comando abaixo no terminal para baixar o arquivo:

wget <link copiado>

Instalando o RankAdmet

Instale o arquivo .whl com o comando:

pip install .\rankadmet*.whl

Aguarde até a conclusão da instalação. Ela pode demorar um pouco.

Verifique se a instalação foi bem-sucedida executando o comando:

rankadmet --version

Uso Diário

Para utilizar o programa se deve ativar o ambiente virtual dele.

Ative a venv com:

<caminho_da_pasta_da_venv>\Scripts\activate

Para desativar, use:

deactivate

Atualizando o RankAdmet

Para atualizar o RankAdmet, siga os passos abaixo com o ambiente virtual ativado:

Remova qualquer arquivo .whl existente no diretório atual:

del *.whl

Acesse esse repositório no GitHub. Na seção "Assets" da última versão do RankAdmet, clique com o botão direito no arquivo .whl e copie o link.

Baixe o arquivo utilizando o comando:

wget <link copiado>

Instale o arquivo baixado com o comando:

pip install \rankadmet*.whl --force-reinstall

Verifique a versão instalada com:

rankadmet --version

Download do Rankadmet

Acesse esse repositório no GitHub. Na seção "Assets" da última versão do RankAdmet, clique com o botão direito no arquivo .whl e copie o link. Em seguida, use o comando abaixo no terminal para baixar o arquivo:

wget <link copiado>

Instalando Dependências

Atualize o gerenciador de pacotes apt com o comando abaixo:

sudo apt update -y
sudo apt upgrade -y

Verifique se tem o Python instalado:

python3 --version

Se não estiver instalado, instale com o comando:

sudo apt install python3

Verifique se o módulo venv está instalado:

python3 -m venv --help

Se não estiver instalado, instale com o comando:

sudo apt install python3-venv

Criação da venv e Instalação do Programa

Crie a venv na pasta /opt/:

python3 -m venv /opt/rankadmet

A ative a venv:

source /opt/rankadmet/bin/activate

Com a venv ativada, instale o arquivo .whl com o comando:

sudo /opt/rankadmet/bin/pip install rankadmet*

Confira a instalação com:

rankadmet --version

Ou apenas:

rankadmet

Uso Diário

Sempre ative o ambiente virtual antes de usar o programa. Use o comando abaixo:

source /opt/rankadmet/bin/activate

Quando terminar, desative o ambiente virtual com o comando:

deactivate

Atualizando

Para evitar conflitos, remova qualquer arquivo .whl existente no diretório atual com o comando:

rm *.whl

Em seguida, acesse o repositório no GitHub. Na seção "Assets" da última versão do RankAdmet, clique com o botão direito no arquivo .whl e copie o link. Use o comando abaixo para baixá-lo:

wget <link copiado>

Ative o ambiente virtual com:

source /opt/rankadmet/bin/activate

Instale o arquivo baixado com o comando:

sudo /opt/rankadmet/bin/pip install rankadmet* --force-reinstall

Verifique a versão instalada com:

rankadmet --version
